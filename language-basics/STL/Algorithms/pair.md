
## header file to include

```
#include <algorithm>
```

## Operations

1. to create a pair using make_pair
2. access the elements with .first and .second
3. swap between two pairs
```
pair1.swap(pair2)
```

4. unpack with tie
```
tie(a, b) = pair2
```
